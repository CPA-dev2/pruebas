# Sistema de Procesamiento de Documentos con IA

Sistema avanzado de OCR y validación de documentos guatemaltecos con capacidades de aprendizaje automático.

## Características

- **OCR Inteligente**: Reconocimiento de texto con PaddleOCR optimizado
- **Validación Multi-nivel**: Validación por palabras clave, patrones y coherencia
- **Scoring Real**: Sistema de scoring basado en múltiples factores (legibilidad, completitud, coherencia)
- **Aprendizaje Automático**: Entrenamiento continuo con muestras etiquetadas
- **Soporte Multi-documento**: DPI (frontal/posterior), RTU, Patentes de comercio
- **Procesamiento Batch**: Procesamiento de lotes de documentos
- **Exportación/Importación**: Guardado y carga de modelos entrenados
